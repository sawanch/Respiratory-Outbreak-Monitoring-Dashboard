## Project Overview

A full-stack web application for tracking and visualizing global COVID-19 statistics with real-time analytics and monitoring.

**Tech Stack:**
- Backend: Spring Boot, MySQL, MongoDB, Redis
- Frontend: JavaScript, Bootstrap, HighCharts
- Infrastructure: AWS EC2, Nginx
